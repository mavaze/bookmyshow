# bookmyshow-k8s-app

```bash
curl https://start.spring.io/starter.tgz \
    -d type=maven-project \
    -d language=java \
    -d bootVersion=2.7.4 \
    -d groupId=io.github.mavaze.bookmyshow \
    -d artifactId=admin-api \
    -d name=admin-api \
    -d packageName=io.github.mavaze.bookmyshow \
    -d dependencies=webflux,actuator,security,oauth2-resource-server,lombok \
    -d javaVersion=17 | tar -xzf -
```
